Let & Const

Quiz

  1. What is the difference between var and let?

    The main difference is that var is function-scoped, while let is block-scoped. Also, contrary to var, let cannot be re-declare within the same scope. Finally, let is not hoisted and is not part of the window object.

  2. What is the difference between var and const?

    const shares the same differences than let within var. Additionally, const cannot be re-assigned.

  3. What is the difference between let and const?

    let can be re-assigned, while const cannot.

  4. What is hoisting?

    Hoisting meants that you can use a hoisted variable ebfore it's being declared, because JavaScript moves all variable and function declarations to the top of the current scope. However, the variable is not initialized, so it will return undefined. While this is true for var, this is not the case with with let and const.
